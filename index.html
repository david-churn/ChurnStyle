Automatic Definitions

Colors

The base document is given an off-white color so form fields will stand out from the background color. Light green is used to split sections of the document and is available by using class="card". Yellow is used for highlighting items when they are hovered over.

Fonts

The header, footer, section titles (h1-h6), and form legend are all preset to include the techno-like font to make them pop off the page.

The regular text uses a readable font because we do not want to go completely over the top. Paragraph lines are limited to make them easier to read.

Options Available

Buttons

There are three different buttons available. All use the oval shape and highlight in yellow when the cursor hovers over them. The reversed button is invoked by using class="rev-btn". The special techno-like button is invoked by using class="tech-btn".

Grid System

The grid system is ready for use. It uses a 12 unit system. All grid units are set to highlight when the cursor hovers over them.

Code example

        <div class="row">
          <div class="grid_unit flex5">flex5</div>
          <div class="grid_unit flex7">flex7</div>
        </div>

Horizontal Navigation Bar

There is a horizontal navigation bar available. By default it stretch across the entire width of the screen allowing you the flexibility to adjust to the requirements. It is defined between <nav> statements using unorder lists to define the main section with one set of drop downs. Drop down items are highlighted with yellow when the cursor hovers over them.

Useful Development Classes

The following classes are useful for invoking the effects built into this style sheet. All of these are simply an additional class added to the html mark up.

    Class Names
  • border - put a border around it.
  • card - define a green background similar to the grid blocks.
  • center - put the text in the middle.
  • greenish - change the background to the theme's green
  • hidden - show an off-color background to highlight where fields will eventually be hidden.
  • logo - put on an image to make a small image out of it. (See below).
  • italicize - use italics to highlight text.
  • sranger - use the heading font to improve the nerdiness of the document.
  • yellowish - change the background to the theme's yellow
